# Developing Geospatial Accessibility Models for Healthcare Facilities to Improve Access for People with Disability.

This project aims to improve healthcare accessibility for people with physical disabilities in the UK by developing a geospatial accessibility model using advanced machine learning techniques. The model integrates publicly available big data sources, including **OpenStreetMap (OSM)**, **WheelMap**, and patient feedback from platforms like **Yelp**, to analyse healthcare facility infrastructure and user experiences, generating accessibility scores that capture both external travel and internal facility constraints.

The model primarily focuses on wheelchair users while remaining adaptable to other mobility impairments, enabling individuals, caregivers, and planners to make informed decisions about accessible healthcare options. Advanced computational methods, such as **Transformer-based masked autoencoders**, are applied to uncover complex patterns in spatial and textual data. The results are visualised through a user-friendly web or mobile application, providing interpretable insights into accessibility across urban environments.

The project is conducted in collaboration with the **International Centre for Evidence in Disability (ICED) at LSHTM** and the **Birkbeck Institute for Data Analytics (BIDA)**, combining expertise in disability research and data science. Engagement with people with disabilities and their caregivers will ensures that the model and tool reflect lived experience. Findings will inform urban planning and healthcare policy by offering data-driven recommendations to improve facility accessibility throughout the UK.

This interdisciplinary research demonstrates the power of combining computational methods, geospatial data, and participatory evaluation to address urgent societal needs in inclusive healthcare access.
