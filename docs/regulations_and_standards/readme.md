# Standards and Regulatory Framework for Defining the Scope of Physical Disability in the UK

## Aprroved Document M
**Approved Document M** is a set of guidelines in the UK that provides practical guidance on building regulations to ensure access and inclusion for all. It is divided into **two volumes**:

**Volume 1**: Covers dwellings and provides guidance on accessibility features.

**Volume 2**: Covers buildings other than dwellings and includes requirements for accessibility.

The document was first published in **March 2015** and has been updated with amendments since then.
